# GUI 增强功能使用说明

## 新增功能概览

本次更新为 Java 项目分析智能体 GUI 添加了以下增强功能：

### 1. Markdown 富文本渲染 📄

**功能描述：**
- Markdown 内容不再以源文件形式显示，而是渲染为类似 Obsidian 的富文本格式
- 支持标题、代码块、列表、链接、粗体、斜体等格式
- 代码块使用等宽字体显示，带有背景色区分

**支持格式：**
- 标题 (`#`, `##`, `###`, `####`)
- 代码块 (` ``` `)
- 行内代码 (`` `code` ``)
- 粗体 (`**text**`)
- 斜体 (`*text*`)
- 无序列表 (`-`, `*`)
- 有序列表 (`1.`, `2.`)
- 引用块 (`>`)
- 链接 (`[text](url)`)

---

### 2. 主题切换 🎨

**支持三种主题：**

| 主题 | 说明 | 适用场景 |
|------|------|----------|
| 亮色主题 | 白色背景，深色文字 | 白天、光线充足环境 |
| 暗色主题 | 深色背景，浅色文字 | 夜间、低光环境 |
| 护眼主题 | 淡绿色背景，深绿色文字 | 长时间阅读、护眼需求 |

**使用方法：**
1. 在工具栏找到 "🎨 主题" 下拉框
2. 选择需要的主题
3. 主题会立即应用到整个界面
4. 设置会自动保存，下次启动时恢复

---

### 3. 字体大小调节 🔤

**支持四档字体大小：**

| 大小 | 基准字体 | 说明 |
|------|---------|------|
| 小 | 10pt | 适合高分辨率屏幕 |
| 中 | 12pt | 默认大小，适合大多数用户 |
| 大 | 14pt | 适合需要更大文字的用户 |
| 超大 | 16pt | 适合视力不佳或远距离观看 |

**使用方法：**
1. 在工具栏找到 "字体" 下拉框
2. 选择需要的字体大小
3. 字体大小会立即应用到所有文本组件
4. 设置会自动保存，下次启动时恢复

## 使用技巧

### 1. 主题选择建议

- **亮色主题**：适合白天使用，文字对比度高，阅读清晰
- **暗色主题**：适合夜间使用，减少蓝光，降低眼睛疲劳
- **护眼主题**：适合长时间阅读，淡绿色背景有助于缓解眼疲劳

### 2. 字体大小选择

- 如果不确定，使用默认的"中"大小
- 在高分辨率屏幕（2K/4K）上，建议使用"大"或"超大"
- 如果感觉文字太大导致每行显示内容过少，可以调小字体

### 3. 快捷键

- `Enter` - 在输入框中按下 Enter 发送问题
- 主题和字体选择支持快速切换，无需重启应用

## 常见问题

### Q: 如何切换回原版 GUI？

**A:** 修改 `run.py`，将导入改回原版：

```python
# 增强版
from gui.main_window_enhanced import main as gui_main

# 原版
from gui.main_window import main as gui_main
```

### Q: 主题切换后内容显示异常？

**A:** 这是正常现象，主题会立即应用到所有组件。如果遇到问题，尝试：
1. 切换到其他主题再切回来
2. 重启应用

### Q: 字体大小改变后布局错乱？

**A:** 字体大小会影响组件高度，但布局会自动调整。如果布局异常：
1. 尝试调整窗口大小
2. 重启应用

### Q: 设置保存在哪里？

**A:** 设置保存在 `config/gui_settings.json`，可以手动编辑该文件修改设置。
