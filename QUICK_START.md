# 快速启动指南

## 🚀 5分钟快速上手

### 步骤1: 安装依赖 (1分钟)

```bash
cd d:\pyproject\klgm1
pip install -r requirements.txt
```

---

### 步骤2: 准备Java项目 (1分钟)

确保你有一个标准的Java项目，例如：

```
my-spring-boot-project/
├── pom.xml
└── src/
    └── main/
        └── java/
            └── com/
                └── example/
                    ├── DemoApplication.java
                    ├── controller/
                    ├── service/
                    └── model/
```

**如果没有现成项目：**
- 从 [Spring Initializr](https://start.spring.io/) 生成一个示例项目
- 或使用GitHub上的开源Spring Boot项目

---

### 步骤3: 配置API密钥 (1分钟)

**方式1: 环境变量（推荐）**
```bash
# Windows PowerShell
$env:OPENAI_API_KEY="sk-your-api-key"

# Windows CMD
set OPENAI_API_KEY=sk-your-api-key

# Linux/Mac
export OPENAI_API_KEY=sk-your-api-key
```

**方式2: 在GUI中配置**
- 启动应用后点击"⚙️ 模型配置"
- 填写API Key、Base URL、Model Name

---

### 步骤4: 启动应用 (30秒)

```bash
python run.py
```

或直接双击 `run.py` 文件。

---

### 步骤5: 开始使用 (2分钟)

1. **选择项目**
   - 点击 "📁 选择项目"
   - 浏览并选择你的Java项目根目录

2. **分析项目**
   - 点击 "🔍 分析项目"
   - 等待10-30秒
   - 查看生成的 `overview.md`

3. **学习知识点**
   - 点击 "📚 下一个知识点"
   - 阅读生成的讲解内容
   - 切换到"问答记录"Tab提问

4. **查看进度**
   - 切换到"📊 学习进度"Tab
   - 了解学习进展

## ❓ 常见问题

### Q: 第一次使用需要什么？
**A:** 
1. Python 3.8+
2. OpenAI API密钥（或其他兼容的LLM服务）
3. 一个Java项目

### Q: 可以用免费的LLM吗？
**A:** 可以！支持：
- Ollama（本地运行，完全免费）
- LM Studio（本地运行）
- 其他兼容OpenAI API格式的服务

### Q: 分析一个项目要多久？
**A:** 
- 首次分析：10-30秒（取决于项目大小和网络速度）
- 知识点讲解：5-15秒/个
- 回答问题：2-5秒/个

### Q: 生成的文件在哪里？
**A:** 在你选择的Java项目根目录下：
```
your-project/
├── overview.md       # 项目分析报告
└── note/             # 知识点讲解
    ├── topic1.md
    └── topic2.md
```
